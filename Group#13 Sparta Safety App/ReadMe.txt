*Sparta Safety App*

"Safety First" is "Safety Always."  
~Charles M. Hayes

This is a mobile app, meant for the safety of SJSU students. This application uses past data to give you information about unsafe incidents across SJSU campus at different times. This app can calculate user's safety percentage and guide them to remain safe. App has a provision for sending notification to every user, when they see any suspicious activity in or around SJSU. Sparta Safety aims at providing safety to the SJSU students because, "Prevention is always better than cure"


Direction for Usage:

1). Android folder contains source code for front end of the app. It also contains the "apk", which must be installed in the mobile phones.

2). Nodejs folder contains the business logic.

3) Data folder contains the JSON data , which must be imported into MonogDb database.
